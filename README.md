# Modular Forecasting Streamlit App (Daily & Monthly)

**IMPORTANT: This is a STREAMLIT application, NOT a Shiny application.**

Upload a CSV, pick your model & frequency (Daily/Monthly), and forecast with best hyperparameters. Download the results!

## Deployment Notes

- **Application Type**: Streamlit (uses `streamlit` library)
- **Entry Point**: `Forecast_ML.py` or `main.py`
- **Dependencies**: See `requirements.txt`
- **NOT Required**: `shiny` package (this is NOT a Shiny app)

## Running Locally

```bash
streamlit run Forecast_ML.py
# or
streamlit run main.py
```
